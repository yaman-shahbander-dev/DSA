# DSA
This repository is all about Data Structures and Algorithms. 
It focuses on the implementation of the different concepts that are in DSA using the PHP language.

## Data Structures ##
* Stacks
* Linear Queues
* Circular Queues
* Trees
* Graphs
* Singly Linked Lists
* Doubly Linked Lists

## Algorithms ##
* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
